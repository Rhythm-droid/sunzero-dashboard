# SunZero Energy Intelligence Dashboard

A real-time energy monitoring and analytics dashboard for SunZero's
Solar and Battery Energy-as-a-Service (SEAS/BEAS) platform.

## Overview

This is a mockup/prototype dashboard demonstrating SunZero's
proposed energy intelligence platform. It serves three user roles:

- **Customer/Stakeholder** — Energy monitoring, savings tracking, carbon impact
- **Investor** — Portfolio performance, ROI tracking, project details
- **Admin** — Fleet monitoring, alerts, site management

## Tech Stack

- HTML5
- TailwindCSS (via CDN)
- Vanilla JavaScript
- Deployed on Vercel

## How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
3. Select a role and sign in

## How to Deploy

1. Push to GitHub
2. Connect repo to Vercel
3. Deploy — done!
